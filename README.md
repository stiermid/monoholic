# monoholic

Minimal dark jekyll theme

## Installation

Add this line to your Jekyll site's `Gemfile`:

```ruby
gem "monoholic"
```

And add this line to your Jekyll site's `_config.yml`:

```yaml
theme: monoholic
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install monoholic
